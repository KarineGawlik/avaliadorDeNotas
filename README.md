# Avaliador de Notas

Grade average calculator developed with HTML, CSS and JavaScript.

🔗 Live project: https://avaliador-de-notas.vercel.app/

## Technologies

- HTML
- CSS
- JavaScript
- Vercel

## Purpose

This project was created to practice DOM manipulation, form handling, basic calculations and dynamic content updates with JavaScript.

## Features

- Add activities and grades
- Calculate final average
- Show approval status
- Dynamic table update
- Simple and responsive interface
